Git for Beginners

Publishing your portfolio to GitHub

01

Install Git

Download Git from git-scm.com and install it. Then set your identity so your commits are signed.

# One-time setup git config --global user.name "EU Ubia" git config --global user.email "you@email.com"
02

Create the repo on GitHub

Go to github.com → New Repository → name it EUbia3.github.io → tick "Public" → Create. This exact name unlocks GitHub Pages hosting.

03

Clone & add your file

Clone the empty repo to your computer, drop in your portfolio HTML file, then stage it.

git clone https://github.com/EUbia3/EUbia3.github.io cd EUbia3.github.io # Copy index.html into this folder git add index.html
04

Commit & push

A commit is a snapshot of your work. Push sends it to GitHub. Your site goes live automatically.

git commit -m "Add portfolio site" git push origin main # Live at: # https://EUbia3.github.io

Pro tip — updating your site

Every time you make changes, just run these three commands. Your live site updates within seconds:

git add . git commit -m "Update: describe what you changed" git push
